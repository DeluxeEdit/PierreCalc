◆ sa_properties_expand_alloc()

const char * sa_properties_expand_alloc ( const char * value)

Expand the given string by replacing property variable reference by the actual variable's value. The syntax of a property variable reference is the following: ${variable-name} where variable-name is the name of a variable.

The string is expanded until there are no change in the given string.

Parameters
valueThe given value to expand.
Returns
Returns a valid string value. Must be free with sa_memory_free() to prevent memory leaks. Returns NULL if no value is defined.

◆ sa_properties_expand_buffer()

sa_error_t sa_properties_expand_buffer ( const char * value,
int * expanded_length,
char * buffer,
size_t buffer_size )

Expand the given string by replacing property variable reference by the actual variable's value. The syntax of a property variable reference is the following: ${variable-name} where variable-name is the name of a variable.

The string is expanded until there are no change in the given string. The expanded_length is updated even if the given buffer is too small.

Parameters
valueThe given value to expand.
expanded_lengthThe length of the expanded value in bytes.
bufferThe output buffer for the expanded value.
buffer_sizeThe size of the output buffer in bytes.
Returns
Returns 0 on success. Returns a non-zero on error.

◆ sa_properties_expand_once_alloc()

const char * sa_properties_expand_once_alloc ( const char * value)

Expand the given string by replacing property variable reference by the actual variable's value. The syntax of a property variable reference is the following: ${variable-name} where variable-name is the name of a variable.

The string is expanded in a single pass. Some property reference may not be expanded after the first pass.

Parameters
valueThe given value to expand.
Returns
Returns a valid string value. Must be free with sa_memory_free() to prevent memory leaks. Returns NULL if no value is defined.
